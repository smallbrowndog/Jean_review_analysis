## KoELECTRA를 활용한 무신사 바지 리뷰 분석
![image](https://github.com/user-attachments/assets/178fbb32-3975-40c0-997d-c514f2e20bcb)

## 1. 개요
### 1-1 무신사 바지 리뷰로 결정한 이유
29CM, 한섬 EQL, 무신사, SSF샵 등의 많은 온라인 쇼핑몰 중에서도 무신사를 선택한 이유는 여러가지가 있다.
우선 품목별 세부 카테고리가 체계적으로 정리되어 있어 사용자들이 원하는 제품을 쉽게 찾을 수 있으며 국내 최대 규모의 온라인 패션 플랫폼 중 하나로 가장 많은 이용자 수와 방대한 양의 리뷰를 보유하고 있기때문이다. 또한 무신사가 익숙한 플랫폼인것도 하나의 이유였다.

많은 의류 카테고리 중에서 바지 카테고리로 정한 이유는 그동안 여러 브랜드의 바지를 구매하면서 만족스럽지 못한 경험도 많았다. 이 때문에 정보를 찾아보던 중 브랜드별 평판이 다르다는 것을 알게 되었다.
하지만 기존의 리뷰들은 대부분 긍정적인 내용만 담고 있어, 브랜드의 실제 품질을 판단하기에는 부족함이 있었다.

내가 겪었던 문제를 해결하기 위해 이번 프로젝트에서는 무신사에 입점한 브랜드들의 바지 리뷰 데이터를 수집하고 이를 기반으로 긍정/부정 예측 모델을 구축하고자 한다.

## 2. 데이터
### 2-1 데이터 수집 계획
이전의 아마존 패션 카테고리 리뷰 분석 프로젝트를 진행할때는 기존에 제공되었던 데이터를 활용하여 프로젝트를 진행했었다.
하지만 해당 데이터는 의류가 아니라고 볼 수 있는 데이터들이 많이 섞여 있었고 전처리 이후 직접 살펴본 의류들도 내가 바라던 의류가 아니었기도 했기에 이번 프로젝트에서는 내가 원하는 데이터를 직접 수집해보고자 하였다.
